LEGO Robot Project

This project was part of my ENED 1100 & ENED 1120 courses where I designed and programmed a LEGO robot.

Project Overview

The project involved designing and programming a LEGO robot to perform specific tasks. In the ENED 1100 course, I built a basic robot to navigate lines on the ground. In ENED 1120, the project evolved to create an advanced LEGO robot using Python. The robot was designed to detect and pick up items based on color and move around obstacles.

Project Goals & Challenges

  • Design a functional LEGO robot using sensors and motors
  • Write code to control the robot's movements and actions (Python)
  • Test and troubleshoot the robot's performance in various environments
  • Challenge: Ensuring the robot accurately detects and picks up objects while avoiding obstacles

Lessons Learned

This project taught me valuable skills in robotics, problem-solving, and programming. I learned how to work with both hardware and software components, and how to refine my programming techniques to improve the robot's functionality. The project also helped me enhance my teamwork and communication skills, as I collaborated with classmates to solve complex issues.
